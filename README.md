# TouchDesigner Custom Operators

This repository contains the base headers for TouchDesigner custom operators. It might contain utilities in the future, but is header-only for now.

## Bazel

I've included a basic Bazel module for reference purposes. I ran an experiment trying to integrate all this together with Bazel, but failed. Eventually, I'll support Bazel for my downstream dependencies, but first, I'll be using CMake for exhaustive support.
